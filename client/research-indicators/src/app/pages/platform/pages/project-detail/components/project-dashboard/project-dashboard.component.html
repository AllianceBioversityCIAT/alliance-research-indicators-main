The figures shown in this dashboard are based on results reported in the STAR platform. Some metrics rely on metadata that is currently available for STAR results. Results imported from TIP, PRMS, and AICCRA are not yet enriched with the same metadata, so the level of detail displayed in certain dashboard visualizations may vary depending on the source platform. We are actively working to make information from these additional systems available with the same level of detail and consistency.
